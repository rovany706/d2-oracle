# D2Oracle
Tool that helps player during a game of Dota 2 using audio hints and providing useful info about current hero builds.

<img src="img/main_window.png"  />

## Features
- Sound hints
- Timers
  - Roshan timer
  - Wisdom rune timer
- Net worth graph
- Dota2ProTracker integration
